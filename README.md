Repositorio de materiales, ejercicios y proyectos del módulo DWES del curso 2025/2026.

## Estructura

```
DWES/

│   ├── index.php
│   └── proyecto/               # Proyecto PHP: Mountain Connect
│       ├── mountain-connect Fase1/

│   ├── teoría/                 # Guías (MVC, rutas, vistas, modelos, sesiones)

│   ├── appPruebaModelos/       # App de prueba — modelos y BBDD
│   ├── proyecto/               # Proyecto Laravel (appProyecto)
│   └── Examen/                 # Proyecto examen Laravel
│
├── Examen Ev1/                 # Examen 1ª Evaluación
│   ├── Plantilla_examen_1EV_php/
│   └── examen_1EV_php Resultado/
```
