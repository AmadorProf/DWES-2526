# DWES — Desarrollo Web en Entorno Servidor

Materiales del modulo DWES del curso 2025/2026. PHP, Laravel y MySQL desde cero hasta proyectos completos.

## Estructura

```
PHP/                      — Bloque 1: PHP
  *.md                    — Teoria: formularios, sesiones, POO, BBDD
  proyecto/               — Proyecto Mountain Connect (Fase 1 y 2)

laravel/                  — Bloque 2: Laravel
  teoria/                 — Guias: rutas, vistas Blade, modelos, sesiones
  appPrueba/              — App de prueba: rutas y controladores
  appPruebaModelos/       — App de prueba: modelos y BBDD
  proyecto/               — Proyecto Laravel final
  Examen/                 — Proyecto examen Laravel

examen-ev1/               — Examen 1a evaluacion (enunciado + plantilla)
examen-ev2/               — Examen 2a evaluacion (enunciado + solucion)
examenes-finales/         — Examenes globales
evaluaciongithub/         — Actividad de evaluacion con Git y GitHub
referencias/              — API REST y otras referencias tecnicas
```

## Como usar estos materiales

Accede a la [pagina web](https://amadorprof.github.io/DWES-2526/) para navegar por todos los apuntes. Los archivos `.md` se pueden leer directamente en GitHub o desde el visor web.

Para ejecutar los proyectos PHP y Laravel necesitas XAMPP o Docker. Las instrucciones de instalacion estan en `PHP/Guia_XAMPP_PHP_VSCode.md`.

## Tecnologias

- PHP 8+
- Laravel 11
- MySQL / SQLite
- Docker / Docker Compose
- Nginx
