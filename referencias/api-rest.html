API REST con Laravel

Guía paso a paso para crear tu primera API profesional

1

Instalación de Laravel

Instala Laravel usando Composer. Asegúrate de tener PHP 8.1+ y Composer instalados en tu sistema.

composer create-project laravel/laravel mi-api
Composer PHP 8.1+
2

Configuración de Base de Datos

Configura tu conexión a la base de datos en el archivo .env. Luego ejecuta las migraciones.

DB_CONNECTION=mysql
DB_DATABASE=mi_api
DB_USERNAME=root
DB_PASSWORD=

php artisan migrate
MySQL PostgreSQL SQLite
3

Crear Modelos y Migraciones

Genera los modelos con sus migraciones correspondientes. Esto creará la estructura de tu base de datos.

php artisan make:model Post -m
php artisan make:model User -m
Eloquent ORM Migraciones
4

Crear Controladores API

Genera controladores específicos para la API con métodos CRUD predefinidos.

php artisan make:controller Api/PostController --api
php artisan make:controller Api/UserController --api
REST CRUD
5

Definir Rutas API

Define las rutas en routes/api.php. Laravel automáticamente añade el prefijo /api a estas rutas.

Route::apiResource('posts', PostController::class);
Route::apiResource('users', UserController::class);
Routing Resource Routes
6

Crear Resources (Transformadores)

Los Resources transforman tus modelos en respuestas JSON estructuradas y consistentes.

php artisan make:resource PostResource
php artisan make:resource PostCollection
JSON Transformadores
7

Validación de Requests

Crea Form Requests para validar los datos entrantes de manera organizada y reutilizable.

php artisan make:request StorePostRequest
php artisan make:request UpdatePostRequest
Validación Seguridad
8

Autenticación (Sanctum)

Implementa autenticación de API usando Laravel Sanctum para tokens de acceso.

composer require laravel/sanctum
php artisan vendor:publish --provider="Laravel\Sanctum\..."
php artisan migrate
JWT Tokens Sanctum
9

Testing de la API

Crea tests automatizados para asegurar que tu API funciona correctamente.

php artisan make:test PostApiTest
php artisan test
PHPUnit Testing
10

Documentación con Scribe

Genera documentación automática de tu API para facilitar su uso.

composer require --dev knuckleswtf/scribe
php artisan vendor:publish --tag=scribe-config
php artisan scribe:generate
Documentación OpenAPI

Mejores Prácticas

Versionado Usa versionado en tus rutas: /api/v1/posts
CORS Configura CORS correctamente en config/cors.php
Rate Limiting Implementa límites de peticiones para proteger tu API
Paginación Usa paginación para colecciones grandes de datos
Códigos HTTP Retorna los códigos de estado HTTP apropiados
Caché Implementa caché para mejorar el rendimiento
